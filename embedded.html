Embedded Systems

We know that in today's fast changing world everyone wants to remain updated about new technologies. So here we provide a certified course on Embedded System using ARM. In this we help students to learn and to have a hands-on experience on this new mbed platform that will help them to get ready for industry.
Understanding the need of the students to start from basics, the training will start from C programming basics and go up to advanced industry level projects.

COURSE CONTENT

The course content is divided along the lines of following :

LED, LED Matrix, 3D Cube Interface

Switch, Digital Sensors Interface

Analog Sensor, Temperature Sensor, Light Sensor Interface

LCD Interface

Accelerometer Interface

Data Acquisition System using Graph

Bluetooth Interfacing

Nrf Interfacing
